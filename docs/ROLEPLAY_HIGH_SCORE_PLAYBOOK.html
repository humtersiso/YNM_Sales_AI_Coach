一、建議學習流程

銷售助手查 RAG 預習 → 設定車型／競品／人設 → 對練（閉卷，畫面不顯示佐證）→ 五維評分
步驟 做什麼
1. 銷售助手 查詢本場競品相關知識（油耗、配備、對戰話術等）
2. 情境設定 選車型、競品、人設、難度；RAG 不足時無法開局
3. 演練中 像真實展間對話回應客戶；畫面不顯示佐證，評分在背後依知識庫對照
4. 完賽 對照五維圖與「未使用策略」再練

覆蓋率稽核：npm run audit:roleplay-rag → 報告見 docs/ROLEPLAY_RAG_COVERAGE_AUDIT.html
HTML 版攻略：npm run docs:roleplay-playbook → docs/ROLEPLAY_HIGH_SCORE_PLAYBOOK.html
本文件競品章節依 npm run dump:roleplay-rag（data/roleplay-rag-playbook-snapshot.json）實測更新。

二、評分怎麼算

項目 說明
總分 五維各 0～20，加總 0～100
等級 S ≥90、A 80–89、B 70–79、C 60–69、D <60
事實依據 僅本場 sectionC RAG 佐證
factCheck 客戶問到且業代與佐證矛盾才重扣；未問的不扣
常見扣分 空泛話術、未用策略、沒有試乘／試算邀約

三、五維拿分公式

1. 同理承接（empathy）

先接住客戶情緒與動機，再進入說明。

「我理解您會擔心 ___，這也是比 ___ 時常先問的。」

2. 論點完整度（structure）

每輪建議四段式（2～4 句內）：

承接疑慮 → 本場 RAG 事實／基準 → 本品差異 → 試乘或試算邀約

3. 事實引用正確（factCheck）

  • 數字、條件須與銷售助手／知識庫一致（完賽評語會指出落差）
  • 不確定優惠：「依本月總部方案為準」
  • 勿引用佐證沒有的引擎、保養金額、競品數字

4. 策略使用（strategy）

依本場「回應重點」（來自 RAG snippet）落地，每 1～2 輪至少用一條。

5. 推進成交（advance）

給具體下一步：試乘時段、油耗試算表、二訪、保留名額。

四、五輪對話節奏（預設 5 輪）

輪次 業代目標
R1 同理 + 確認關注重點（對齊 RAG 第一條佐證）
R2 引用佐證事實 + 基準／試算
R3 本品差異（不攻擊競品）
R4 補強客戶仍猶豫的維度
R5 明確下一步（試乘／試算／二訪）

3 輪練習：承接 → 事實試算 → 試乘邀約。

五、依人設調整（P-01～P-05）

人設 高分打法
P-01 理性分析 基準、公式、試算；少形容詞
P-02 情感品牌 試乘體感、服務；數字簡短
P-03 預算敏感 方案透明 + 總持有成本試算
P-04 猶豫從眾 小步驟決策（先試算、先試乘）
P-05 深度研究 說明來源與測試條件；不瞎掰；每輪結尾給可驗證下一步（試駕路線、試算表、TNCAP 簡報）

P-05 × Tucson L 實測提醒：factCheck 常達標，但 empathy／advance 易偏低（總分可能卡在 60 以下）。

  • 同理：先承認「網路比較表很多、數字各說各話」，再說「我們用同一套試算表跟您對一次」。
  • 推進：不要只丟規格，要給具體時間（「週六 10 點試 30 分鐘市區＋快速道路，順便帶保養試算表」）。

六、禁止說法（地雷）

  • 攻擊競品品質
  • 保證一定比競品省油
  • 未查證的數字或論壇謠言
  • 承諾未公布折扣
  • 貶低客戶選擇

七、開練前 60 秒檢查清單

  • 已在銷售助手預習本場競品
  • 知道客戶人設與難度
  • 心裡記住銷售助手查過的重點（練習畫面不會顯示答案）
  • 準備一個試算或試乘的具體說法

八、完賽後自我檢核

維度 自問
同理 第一輪有先認同疑慮嗎？
完整 每輪有承接→事實→價值→引導嗎？
事實 數字能否對上本場佐證？
策略 「未使用策略」是否為空？
成交 最後一輪有具體下一步嗎？

九、分數目標

等級 總分 意義
S 90+ 可作示範
A 80+ 穩定上線
B 70+ 對照佐證再練
C 以下 <70 降難度練結構

十、依競品預習（RAG 實測摘要）

以下為 2026-06 本機開局實測 常命中主題；實際對練以當次 RAG 為準。
預習請在 銷售助手 用「X-TRAIL vs {競品}」查詢，對練畫面不會顯示答案。

Toyota RAV4（來源：`RAV4 X-TRAIL VS RAV4改款_20260327.pdf`）

客戶常問：論壇說 RAV4 油耗比較漂亮，十年用車成本怎麼算都不對。

主題 可引用方向（須與銷售助手一致）
十年十萬公里總成本 旗艦對旗艦：X-TRAIL 約 141.6 萬 vs RAV4 約 171.1 萬，差約 29.5 萬；領航對尊爵+ 差約 31.4 萬
油耗誤區 RAV4 旗艦 WLTC 24 km/L、X-TRAIL 旗艦 16 km/L，但車價／折扣／稅金／油電電池保固後，總持有成本仍偏向 X-TRAIL
內裝／空間 冷氣：旋鈕 vs RAV4 按鍵（盲操）；玻璃：雙層隔音 vs 單層（時速 50 約差 3 分貝）；行李 647 L vs 514 L；後座 4/2/4 ＋滑椅 vs 6/4
配備／促案 同級比 HUD、主機尺寸、購車金等（見對戰 PDF 配備表）；優惠以當月總部方案為準

高分話術骨架：

「您算十年成本很專業。RAV4 帳面油耗確實漂亮，但我們用同一張試算表把車價、折扣、稅、油資跟電池保固加總，很多客戶會發現 X-TRAIL 反而省將近三十萬。週末我帶您試乘，順便對照這張表，您看數字比較踏實。」

Honda CR-V（RAG 常混雜 Sportage 對戰語料）

客戶常問：國產價能不能有進口質感？實際開起來、保養差在哪？

主題 可引用方向
定位 X-TRAIL ICE：進口質感＋國產車價；e-POWER 線：獨家電驅
引擎／NVH 1.5T VC-TURBO 可變壓縮比；雙前座隔音玻璃、NVH 強化
保養（若 RAG 命中 Sportage 段落） X-TRAIL 回廠保養約 2～5 千；部分韓系競品約 1～2 萬、零件約 1.5 倍——僅在客戶問到保養／零件時再帶，勿硬扯 CR-V
試駕引導 數據僅供參考 → 邀請試駕，用實際動力／油耗體感說服

高分話術骨架：

「CR-V 口碑好，您會比到保養跟質感很合理。X-TRAIL 這代把隔音玻璃、配備拉到這個價位帶，建議您先試 20 分鐘，感受 NVH 跟加速，再跟 CR-V 您心裡的分數對照。」

Hyundai Tucson L（來源：`TUCSON L 小改款對應話術_20250410_V4.pdf` 等）

客戶常問：跟 Tucson L 比，動力跟後續養護差多少？

主題 可引用方向
車系／售價 小改款編成與各級距售價（例：X-TRAIL 旗艦 122.9 萬 vs Tucson L 油電 GLTH-B 123.9 萬、汽油 GLT-B 112.9 萬）
動力 X-TRAIL：1.5T VC-TURBO＋輕油電；Tucson L 油電：1.6T＋HEV、汽油 1.6T
安全 TNCAP：X-TRAIL 成人保護 96%／5 星；Tucson L 成人 72%／4 星（客戶問安全時再引，附「以官方公布為準」）
油電／積碳（若 RAG 帶出 Sportage 段落） 傳統 HEV 引擎／馬達切換、Tucson L 舊款油電引擎品情——只在客戶問油電可靠性時使用

P-05 加強：除 fact 外，每 2 輪要有一次可執行邀約（試算表＋試駕時段），避免只答規格。

高分話術骨架：

「Tucson L 小改款您有研究。動力型式不同，與其看錶面數字，不如週六我準備同一段路試 ICE 跟 e-POWER，再附五年保養試算，您帶著比較表回去跟家人討論。」

Mitsubishi Outlander（RAG 專章較少，常命中 Sportage／Territory 語料）

客戶常問：跟 Sportage 比，X-TRAIL 技術穩定性如何？（開場可能帶韓系油電積碳話題）

策略 說明
對齊客戶問題 客戶提 Sportage／HEV 積碳 → 用 RAG 中 HEV 舊世代、引擎積碳、保養費 段落回應，再拉回 X-TRAIL ICE／e-POWER 差異
本品賣點 VC-TURBO、e-POWER 平順、雙層隔音、RR-AEB／PFCW（RAG 有命中時）
預習建議 銷售助手查「X-TRAIL vs Outlander」＋「X-TRAIL ICE 穩定性」；若開局佐證偏 Sportage，以客戶實際追問為準作答

高分話術骨架：

「論壇對韓系油電的討論我懂。X-TRAIL 這次 ICE 走 VC-TURBO 輕油電，跟傳統 HEV 路線不同；與其看文章，不如試乘時我們看儀表與實際油耗，您再判斷穩定性。」

KIA Sportage（來源：`SPORTAGE X-TRAIL 對戰 SPORTAGE 對戰話術策略_260508.pdf`）

客戶常問：外型好看，但油電是舊款？保養會不會很貴？

主題 可引用方向
產品世代 台灣上市車型為國外上市 兩年以上 的舊產品（韓國 2024、歐洲 2025/6）
油電引擎 沿用 Tucson L 油電混合舊款引擎；品情：引擎積碳、儀表亮燈；HEV 切換頓挫、零件損耗
保養／零件 回廠保養約 1～2 萬 vs X-TRAIL 2～5 千；零件費用約 1.5 倍
本品對比 e-POWER 馬達直驅較平順；e-4ORCE 四驅；2023 年起標配 雙層隔音；中高規 RR-AEB、PFCW
油錢 10 年 10 萬公里，Sportage 油電省油約 4 萬——需搭配頂規價差一併說明，避免只講單面

高分話術骨架：

「Sportage 設計確實搶眼。若您在意長期養車，我們用原廠保養表算五年，再試 e-POWER 的加速跟靜音；很多客戶比完發現，省下的油錢未必抵過保養跟配備級距差。」

  • 開局時 fetchRoleplayRagContext 檢索三類語料庫；有效佐證 < 2 則拒絕開局

  • 評分 LLM 與 fallback 皆以本場 sectionC 為 factCheck 依據
  • QA 詳解：npm run docs:roleplay-qa-drill（RAG 自動產題）
